# multi-page

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm serve
```

### Compiles and minifies for production
```
npm build
```

### Lints and fixes files
```
npm lint
```

```shell
.
├── App.vue
├── assets
│   └── logo.png
├── components
│   └── HelloWorld.vue
└── pages
    ├── pageA
    │   ├── index.html
    │   ├── main.js
    │   ├── router.js
    │   ├── store.js
    │   └── views
    │       ├── About.vue
    │       └── Home.vue
    └── pageB
        ├── index.html
        ├── main.js
        ├── router.js
        ├── store.js
        └── views
            ├── About.vue
            └── Home.vue
```
